## Project Setup

- Primeiro instale o projeto utilizando o comando abaixo:

```sh
npm install
```

- Compilando o projeto e criando ambiente local para desenvolvimento:

```sh
npm run dev
```

- Compilando e minificando para Produção:

```sh
npm run build
```
